Git tiene un flujo con 3 estados:

  • Directorio de trabajo: Aquí es donde trabajas con tus archivos.
  • Staging area: Aquí agregas tus archivos manipulados y los preparas para su guardado, es un intermediario.
  • Repositorio: Una vez que decidas hacer los cambios, estos archivos pasarán aquí.
